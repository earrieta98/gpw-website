# Server Rack Assembly Plant — Zone-by-Zone Implementation Checklist
## Global Precision Works (GPW) — Monterrey, Mexico

**Purpose:** Detailed implementation guide for each facility zone
**Format:** Pre-construction, construction/installation, and pre-operations checklists

---

## ZONE 1: RECEIVING & INCOMING INSPECTION

### Pre-Construction Phase

#### Layout & Space Planning
- [ ] Confirm total area: 1,200–1,800 sq ft
- [ ] Dock placement: 2–3 bays, each 14 ft wide × 10 ft high
- [ ] Dock apron: Minimum 30 ft clearance from property line
- [ ] Receiving office: 200–300 sq ft (separate from dock floor)
- [ ] Inspection work area: 400–600 sq ft (climate-controlled)
- [ ] Material staging area: 200–300 sq ft (pallets, bins, temporary storage)
- [ ] Aisle width: Minimum 8 ft for forklift and pallet jack movement

#### Utility Planning
- [ ] Electrical: 480V three-phase feed for dock levelers, scales, future conveyor
- [ ] Lighting: High-bay lights (minimum 300 lux at dock, 500 lux at inspection workstations)
- [ ] Data/Network: CAT6 drops to receiving office (3–4 outlets)
- [ ] Compressed air (if pneumatic tools): 1–2" main line with branch feeds

#### Environmental Design
- [ ] HVAC: Standard warehouse conditions acceptable (temperature 15–27°C, humidity 30–70% RH)
- [ ] Doors: 10 ft high × 14 ft wide primary entrance to dock; secondary personnel door to facility
- [ ] Flooring: Industrial epoxy or concrete; no ESD required (materials in packaging)
- [ ] Drainage: Concrete floor sloped toward drains if wet washing of dock area planned

### Construction/Installation Phase

#### Dock Infrastructure
- [ ] Install dock levelers (2–3 units, one per bay)
  - [ ] Verify load capacity: Minimum 25,000 lbs (typical loaded pallet)
  - [ ] Test hydraulics and safety mechanisms
  - [ ] Set height: 48–52" above grade
- [ ] Install vehicle restraint systems (dock locks)
  - [ ] Test for 53 ft trailers
  - [ ] Verify automatic release mechanisms
- [ ] Install dock seals/bumpers
  - [ ] Weather protection (reduces heat loss/infiltration)
- [ ] Forklift charging station
  - [ ] 208V or 480V feed for battery charger
  - [ ] Dedicated safe area away from traffic

#### Receiving Office Setup
- [ ] Install workstations (2 desks with computers, phones)
- [ ] Install filing/documentation storage (shelving, cabinets)
- [ ] Adequate lighting: Minimum 300 lux
- [ ] Climate control: Separate small HVAC unit or damper control from main facility

#### Inspection Area Equipment
- [ ] Install inspection workbenches (6–8 stations)
  - [ ] Size: 4 ft × 2.5 ft per station
  - [ ] Height: 30–36" adjustable if possible
  - [ ] Electrical: 120V outlets every 8 ft for test equipment
- [ ] Flooring: Non-slip industrial epoxy (forklift and personnel safety)
- [ ] Storage racks (for component bins, staging pallets)
  - [ ] Capacity: 10,000–15,000 lbs typical
  - [ ] Height: 8–10 ft (maximize vertical space)

#### Electrical & Power
- [ ] 480V three-phase feed: 100 amp minimum
  - [ ] Branch circuit for dock levelers
  - [ ] Branch circuit for future conveyor/automation
- [ ] 120/208V distribution: 20–30 amp circuits (minimum 3 circuits)
- [ ] Emergency stop buttons at dock areas
- [ ] Adequate lighting circuits (separate from equipment power)

#### Safety & Compliance
- [ ] Install safety signage
  - [ ] "Forklift Traffic" warnings
  - [ ] "Watch for Pedestrians" signs
  - [ ] Load capacity markings
- [ ] Yellow floor tape: Mark dock boundaries, pedestrian pathways
- [ ] Fire extinguishers: Type ABC, placed every 75 ft
- [ ] First aid kit: Accessible location
- [ ] Emergency eyewash station (if chemicals handled)

### Pre-Operations Phase

#### Equipment & Tools
- [ ] Scales (weight verification)
  - [ ] Floor scale: 5,000–10,000 lbs capacity, ±1 lb accuracy
  - [ ] Hanging scale: 500 lbs capacity for spot checks
- [ ] Barcode/RFID scanners (4–6 units for data entry)
- [ ] Computer terminals (3–4) with WMS or inventory software
- [ ] Receiving software: Configured and tested
- [ ] Label printers (2–3) with thermal labels
- [ ] Test equipment (multimeters, continuity testers, insulation testers)
  - [ ] Calibration verified (within last 12 months)
- [ ] Storage bins/totes (categorized by component type)
- [ ] Pallets: Wooden or plastic, rated for truck loading
- [ ] Pallet jacks (3–4 units)
- [ ] Forklifts (dedicated to receiving area during intake hours)

#### Documentation & Systems
- [ ] Receiving checklist template (physical and digital)
- [ ] Incoming inspection SOP (Standard Operating Procedure)
- [ ] Component database entry template
- [ ] Barcode label printer setup and testing
- [ ] Inventory management system configured
- [ ] Rework/rejection procedure documented
- [ ] Customer contact database (vendor contacts for quality issues)

#### Personnel & Training
- [ ] Assign receiving supervisor (1 FTE)
- [ ] Hire receiving technicians (2–3 FTE)
- [ ] Assign quality inspectors (1–2 FTE)
- [ ] Training program:
  - [ ] WMS/inventory system training
  - [ ] Component identification (learn major assemblies, chassis, PSUs, networking equipment)
  - [ ] ESD awareness (even though not required in packaging, technicians should know protocols)
  - [ ] Forklift safety (if operating lifts)
  - [ ] OSHA general industry compliance
- [ ] Certification/badge system for trained personnel
- [ ] Cross-training: At least one backup for supervisor role

#### Pre-Operations Testing
- [ ] Dock leveler operation test (loaded pallet)
- [ ] Scale accuracy verification
- [ ] Barcode scanner calibration
- [ ] Computer system connectivity (network, power, backup)
- [ ] HVAC function (temperature/humidity readings at dock)
- [ ] Lighting adequacy (photometer check, minimum 300 lux)
- [ ] Fire suppression access (clear pathways to extinguishers)
- [ ] Emergency procedures review (with staff)

#### Metrics & Baseline Data
- [ ] Receiving cycle time baseline: Target ≤30 min per pallet (unload, inspect, enter system)
- [ ] Defect rate baseline: Target ≤2% components requiring rework
- [ ] On-time acceptance rate: Target ≥98%
- [ ] Documentation accuracy: Target 100% (all items logged correctly)

## ZONE 2: STORAGE & KITTING

### Pre-Construction Phase

#### Layout & Space Planning
- [ ] Total area: 2,000–3,500 sq ft
  - [ ] Component storage: 1,200–2,000 sq ft
  - [ ] Kitting stations: 600–1,000 sq ft
  - [ ] WIP staging: 200–500 sq ft
- [ ] Component storage layout:
  - [ ] Small-part shelving: Organize by type (connectors, fasteners, cables)
  - [ ] Pallet racking: 6–10 positions for bulk storage
  - [ ] Lockable cabinets: High-value components (critical circuit cards, control modules)
- [ ] Kitting work area: Isolated section with good visibility, clear of high-traffic aisles
- [ ] Staging/layout area: Space for kitting prep before delivery to assembly line
- [ ] Forklift circulation: Minimum 10 ft aisles for movement

#### Utility Planning
- [ ] Electrical: 480V feed (40–60 amp) for future automation (conveyor, sorting equipment)
- [ ] 120V circuits: 15–20 amp (minimum 4 circuits) for workstations
- [ ] Lighting: 300–500 lux (bright, to facilitate reading labels, BOM verification)
- [ ] Data/Network: CAT6 drops to each kitting station (barcode scanner, label printer)
- [ ] Climate control: HVAC connection (temperature 18–27°C, humidity 40–60% RH)

#### ESD Design
- [ ] Flooring: Plan for dissipative ESD flooring (10⁶–10⁹ Ω)
- [ ] Grounding: Central ground bus location identified
- [ ] Workstation layout: Position near grounding conductor access points

#### Ergonomics & Layout
- [ ] Kitting station design: U-shaped workstations with component bins within arm's reach
- [ ] Work surface height: 30–36" (ergonomic for standing 8-hour shifts)
- [ ] Aisle spacing: Clear pathways between storage and kitting stations

### Construction/Installation Phase

#### Flooring
- [ ] ESD flooring installation
  - [ ] Material: Dissipative vinyl tile (DVT) or conductive vinyl tile (CVT)
  - [ ] Resistance testing: Verify 10⁶–10⁹ Ω (ANSI/ESD 7.1 standard)
  - [ ] Sealing: Any seams or damaged areas must be sealed/repaired
- [ ] Floor markings: Yellow tape for workstation boundaries, component bins

#### Storage Infrastructure
- [ ] Small-parts shelving installation
  - [ ] Type: Adjustable shelving, ANSI-rated for weight
  - [ ] Capacity: Plan for 15–20 lbs per shelf (components are lightweight)
  - [ ] Labeling: All shelves marked with part number, location code
- [ ] Pallet racking installation
  - [ ] Type: Selective pallet racks (standard, accessible)
  - [ ] Capacity: 2,000–3,000 lbs per position
  - [ ] Safety: Cross-bracing, load announcements visible
- [ ] Lockable cabinets for high-value items
  - [ ] Type: Steel or composite, weather-resistant
  - [ ] Location: Central, visible to supervising personnel

#### Kitting Workstations (4 Stations)
- [ ] Each station size: 150–200 sq ft
- [ ] Workbench: 48" W × 30" D × 34" H (height-adjustable if possible)
- [ ] Electrical: 120V outlets (2 per station for scale, label printer, computer)
- [ ] Component bins: Organized tray system (8–12 bins per station, labeled by assembly type)
- [ ] Shelving above workstation: For staging kits before hand-off to assembly
- [ ] Seating: Ergonomic chair (if sitting work involved in any tasks)
- [ ] Documentation stand: Vertical display for BOM, work instructions, assembly diagrams

#### Grounding & ESD Infrastructure
- [ ] Central grounding rail installed (copper or aluminum, ≥6 AWG)
- [ ] Grounding conductors bonded from rail to each workstation
- [ ] Test points installed (for quarterly resistance verification)
- [ ] Grounding resistance verified: <1 Ω to facility main ground
- [ ] Wrist strap attachment points at each workstation

#### Lighting & Climate
- [ ] High-bay or track lighting (minimum 400 lux for component identification)
- [ ] HVAC ducts routed to kitting/storage area
- [ ] Temperature/humidity sensor installed (for compliance monitoring)
- [ ] Return air grilles positioned (away from main aisles to minimize dust)

#### Electrical & Power Distribution
- [ ] 120V distribution panel: 200–300 amp service minimum
  - [ ] Branch circuits: 20 amp for workstations (minimum 4 circuits)
  - [ ] 15 amp circuits for general area lighting/convenience outlets
- [ ] Circuit breakers labeled
- [ ] Ground fault circuit interrupters (GFCI) on all wet-area outlets (if applicable)

### Pre-Operations Phase

#### Equipment & Tooling
- [ ] Workstations equipped:
  - [ ] Barcode scanners (2–3 wireless units)
  - [ ] Label printers (thermal, Zebra or equivalent)
  - [ ] Weight scale (per station or shared central location)
  - [ ] Computer terminals (1–2) running inventory management system
- [ ] Component bins/trays: Organized, labeled with SKU, location code, quantity minimum
- [ ] Packing materials:
  - [ ] ESD-safe tote boxes (plastic, conductive coating)
  - [ ] Anti-static bags (if components removed from original packaging)
  - [ ] Packaging tape, labels, documentation
- [ ] Hand tools (if needed for kit verification)
  - [ ] Screwdrivers, pliers, testing tools
  - [ ] Component gauges/fit verification tools

#### ESD Compliance
- [ ] All personnel assigned wrist straps (reusable, with continuity verification)
- [ ] Wrist strap verification equipment installed (at least 1 station)
- [ ] Grounding mats placed at each workstation (conductive, bonded to ground)
- [ ] Static-dissipative gloves available (if components require protection)
- [ ] ESD training materials available (posters, handouts)

#### Software & Systems
- [ ] Inventory management system:
  - [ ] Component database loaded (part numbers, locations, quantities)
  - [ ] Kitting templates configured (BOM linked to assembly orders)
  - [ ] Barcode system operational
- [ ] Label printer configuration:
  - [ ] Kit labels designed and tested
  - [ ] Barcode labels for tote boxes
- [ ] Receiving integration: Kitting module linked to received inventory

#### Personnel & Training
- [ ] Assign material planner/supervisor (1 FTE)
- [ ] Hire kitting technicians (2–3 FTE)
- [ ] Training program:
  - [ ] System operation (WMS, barcode scanning, label printing)
  - [ ] BOM reading and verification
  - [ ] Component identification (critical for catching wrong/defective items)
  - [ ] ESD awareness and wrist strap use
  - [ ] Ergonomics and safe lifting
  - [ ] Quality standards (zero-defect kitting philosophy)
- [ ] Certification: All personnel must pass system competency test

#### Pre-Operations Testing
- [ ] ESD flooring resistance test (ANSI/ESD 7.1): Verify ≤10⁹ Ω
- [ ] Grounding continuity: All workstations bonded to ground (<1 Ω)
- [ ] Barcode system: Scan multiple labels, verify data entry
- [ ] Label printer: Print quality verification (barcodes readable, labels adhere)
- [ ] Weight scale: Calibration verification (±1 oz accuracy)
- [ ] Inventory system: Test full cycle (receive component → kit → deliver)
- [ ] Lighting: Photometer check (minimum 400 lux at workstations)
- [ ] Climate: Temperature/humidity logger (verify 18–27°C, 40–60% RH)

#### Metrics & Baseline Data
- [ ] Kitting time per order: Target ≤30 min (depending on complexity)
- [ ] Accuracy rate: Target 100% (zero incorrect components in kits)
- [ ] ESD incidents: Target zero
- [ ] Material availability: Target >98% (components in stock when needed)
- [ ] Inventory turnover: Baseline rate (track trends for optimization)

## ZONE 3: SUB-ASSEMBLY AREA

### Pre-Construction Phase

#### Layout & Space Planning
- [ ] Total area: 1,500–2,500 sq ft
  - [ ] Cable assembly benches: 4–6 stations × 100–150 sq ft each
  - [ ] Component pre-assembly: 2–3 stations
  - [ ] Sub-assembly test/QA: 200–300 sq ft
  - [ ] Material staging/WIP: 200–300 sq ft
- [ ] Workstation arrangement: Consider cable routing tables, solder station placement
- [ ] Aisle spacing: Minimum 8 ft (allow tech movement, material carts)
- [ ] Ventilation: Plan for solder fume extraction (if soldering is part of sub-assembly)

#### Utility Planning
- [ ] Electrical: 480V three-phase feed (60–80 amp minimum)
  - [ ] Power for soldering stations (if used): 20 amp / 120V per station
  - [ ] Pneumatic tools: 1/2" or 5/8" air line from facility compressor (if available)
  - [ ] 120V circuits: Minimum 4 circuits for workstations
- [ ] Lighting: 500 lux (bright, critical for cable work and soldering)
- [ ] Data/Network: CAT6 drops for future barcode scanners, documentation systems
- [ ] Compressed air: If pneumatic crimpers used, plan main line with branch feeds
- [ ] Fume extraction: Ducting for solder fumes (if applicable)

#### ESD Design
- [ ] Flooring: Plan for conductive/dissipative ESD flooring (10⁶–10⁹ Ω)
- [ ] Grounding: Central ground bus access points identified near each workstation
- [ ] Work surface: Conductive or dissipative mats

#### Safety & Environmental
- [ ] Fire suppression: This zone may handle solder/flux, plan for Class C fire risk
- [ ] Chemical storage: If flux/solvents used, plan safe storage location
- [ ] Ventilation: Fume extraction design (capture at source, not recirculate to assembly floor)
- [ ] HVAC: Temperature 18–27°C, humidity 40–60% RH

### Construction/Installation Phase

#### Flooring & ESD
- [ ] ESD flooring installation (conductive or dissipative)
  - [ ] Verify resistance: 10⁶–10⁹ Ω (ANSI/ESD 7.1)
  - [ ] Sealing: Repair any damage immediately
- [ ] Grounding rails installed (copper, ≥6 AWG)
- [ ] Workstation bonding: Conductors from central rail to each work surface

#### Workstations & Equipment Mounting
- [ ] Cable assembly benches (4–6)
  - [ ] Type: Heavy-duty workbenches with cable routing guides
  - [ ] Height: 30–36" adjustable
  - [ ] Cable spool racks: Mounted above or beside benches
  - [ ] Cable testing equipment access: Organized storage nearby
- [ ] Soldering stations (if applicable, 2–3 stations)
  - [ ] Type: Precision soldering station with temperature control
  - [ ] Mounting: Secure on workbench, 120V dedicated outlet
  - [ ] Fume extraction: Local capture hood, ducted to outside or carbon filter
- [ ] Component pre-assembly stations (2–3)
  - [ ] Fixture/jig storage nearby
  - [ ] Parts staging shelves
- [ ] Test & QA station
  - [ ] Continuity tester, multimeter, insulation tester mounted/accessible
  - [ ] Cable spool storage organized by type/gauge

#### Pneumatic & Air Systems
- [ ] Main compressor: If facility-wide, ensure capacity (typically 5–10 HP for assembly plant)
- [ ] Air line routing: 1/2" or 5/8" copper/aluminum main line with branch feeds to sub-assembly
- [ ] Regulators & filters: Pressure regulator, moisture trap, oil removal filter
- [ ] Hose storage: Coiled and labeled by task (crimping, etc.)

#### Lighting & Climate Control
- [ ] High-intensity task lighting at cable benches (500–750 lux minimum)
- [ ] Soldering station lighting: Additional focal lighting
- [ ] HVAC: Separate or dedicated zone (temperature 18–27°C, humidity 40–60% RH)
- [ ] Air circulation: Ensure fume extraction doesn't create cold spots
- [ ] Temperature/humidity sensors: Data logging capability

#### Electrical & Power
- [ ] 120/208V distribution panel: 200 amp minimum
  - [ ] Dedicated circuits for each soldering station (20 amp)
  - [ ] Circuits for test equipment (15–20 amp)
  - [ ] Lighting circuits (separate from equipment)
- [ ] Compressed air outlet: 1–2 stations (quick-connect fittings)
- [ ] Emergency stop button: Accessible location
- [ ] Circuit breakers: Labeled and color-coded

### Pre-Operations Phase

#### Equipment & Tools
- [ ] Cable crimpers
  - [ ] Manual: 2–3 units (for non-high-volume crimping)
  - [ ] Semi-automatic: 1–2 units (if higher volume)
  - [ ] Insulation stripper: 2–3 units
- [ ] Soldering equipment (if used)
  - [ ] Soldering irons: 2–3 units with temperature control
  - [ ] Solder: Lead-free, per IPC/WHMA standards
  - [ ] Flux: Rosin-based or lead-free equivalent
  - [ ] Solder sucker, desoldering braid, wire strippers
- [ ] Cable testing equipment
  - [ ] Continuity tester: 2–3 units
  - [ ] Multimeter: 2 units (DMM, 20 MΩ minimum input impedance)
  - [ ] Insulation tester: 1 unit (500V, 1000V range)
- [ ] Work fixtures & templates
  - [ ] Cable routing templates (custom per assembly type)
  - [ ] Component holders/clamps
  - [ ] Wire measurement gauges (verify correct wire gauge per specification)
- [ ] Hand tools
  - [ ] Screwdrivers, pliers, cutters, tweezers (full set)
  - [ ] Torque wrench (if torque specs required)
  - [ ] Safety glasses, work gloves, ESD gloves

#### Cable & Component Materials
- [ ] Cable inventory organized by type
  - [ ] Power cables (various gauges, colors)
  - [ ] Data cables (twisted pair, coaxial, fiber)
  - [ ] Signal cables (shielded, unshielded)
- [ ] Connectors: Organized by type (D-sub, RJ45, circular, custom)
- [ ] Terminals, crimps, ferrules: Properly sized for cable gauge
- [ ] Shrink tubing: Color-coded by size/temperature rating
- [ ] Labels & marking supplies: Heat-shrink labels, permanent markers

#### ESD Compliance
- [ ] Anti-static mats at all workstations (bonded to ground)
- [ ] Wrist straps assigned to all personnel
- [ ] Wrist strap verification equipment (continuity checker)
- [ ] ESD gloves available (for sensitive connector handling)
- [ ] Grounding continuity verified: <1 Ω at each workstation

#### Documentation & Systems
- [ ] Sub-assembly work instructions (one per assembly type)
  - [ ] Cable gauge specifications
  - [ ] Connector types and pinouts
  - [ ] Termination procedures with images/diagrams
  - [ ] Testing acceptance criteria
- [ ] Routing templates (physical or digital)
- [ ] Quality checklist: Solder joint quality, connector seating, cable labeling
- [ ] Traceability system: Serial numbers, batch numbers on completed harnesses

#### Personnel & Training
- [ ] Assign sub-assembly supervisor (1 FTE)
- [ ] Hire sub-assembly technicians (3–6 FTE, depending on complexity)
- [ ] Training program:
  - [ ] Cable gauge identification and purpose (power, data, signal)
  - [ ] Proper crimping technique (force, pressure, connector seating)
  - [ ] Soldering best practices (if applicable) — per IPC-A-620
  - [ ] Electrical testing and safety
  - [ ] ESD awareness and grounding procedures
  - [ ] Quality standards (zero-defect mindset)
  - [ ] Work instruction reading and following
- [ ] Certification: All personnel must pass practical test (crimp quality, solder joint quality, test accuracy)

#### Pre-Operations Testing
- [ ] Cable crimper force verification: Pull-test crimped terminals
  - [ ] Target: Crimp should not slip or pull out
  - [ ] Sample testing on first 10 crimps per day
- [ ] Soldering quality audit (if applicable):
  - [ ] Visual inspection: No cold joints, adequate solder flow, clean flux residue
  - [ ] Continuity test: Verify all connections
- [ ] Electrical testing procedure:
  - [ ] Continuity: All conductors should show <1 Ω resistance
  - [ ] Insulation: Minimum 500V insulation resistance between conductors and shield
- [ ] ESD flooring resistance: Verify 10⁶–10⁹ Ω (ANSI/ESD 7.1)
- [ ] Grounding: All workstations <1 Ω to ground
- [ ] Lighting: Photometer check (minimum 500 lux at work surface)
- [ ] Climate: Temperature/humidity logger (verify 18–27°C, 40–60% RH, stable)
- [ ] Fume extraction (if soldering): Test airflow direction, confirm fumes captured

#### Metrics & Baseline Data
- [ ] Cable assembly time per harness: Baseline (varies by complexity)
- [ ] Quality rate: Target >99% first-pass acceptance
- [ ] Defect types tracked: Improper crimps, solder issues, wrong gauge, connector mismatch
- [ ] Test pass rate: Target 100% (electrical testing)
- [ ] ESD incidents: Target zero
- [ ] Rework rate: Target <1%

## ZONE 4: MAIN ASSEMBLY (U-SHAPED ASSEMBLY LINE)

### Pre-Construction Phase

#### Layout & Space Planning
- [ ] Total area: 2,500–4,500 sq ft (for 5–20 racks/day with 8–15 workstations)
- [ ] U-line configuration:
  - [ ] Entry point: Frame assembly station
  - [ ] Workstations (8–15): Arranged in U-shape for efficient flow
  - [ ] Exit point: Staging for burn-in testing
  - [ ] Return path for material (input side): Kitted materials delivery
- [ ] Workstation spacing:
  - [ ] Each station: 150–250 sq ft (includes staging, tool storage, documentation)
  - [ ] Aisle width (between stations): 5–10 ft (accommodate material carts, technician movement)
  - [ ] Supervisor station: Central location with visibility of all stations
- [ ] Material flow: Entry → Station 1 → Station 2 → ... → Station N → Exit
- [ ] WIP (work-in-process) management:
  - [ ] Staging areas at entry and exit
  - [ ] Buffer storage between stations (minimize WIP, maintain flow)

#### Utility Planning
- [ ] Electrical: 480V three-phase feed (100+ amp)
  - [ ] Power tools (pneumatic/electric): 15–20 amp / 120V circuits (minimum 8 circuits)
  - [ ] Workstation computers: 15 amp / 120V (minimum 4 circuits)
  - [ ] Lighting: Separate circuits (not shared with equipment)
- [ ] Compressed air: If pneumatic screwdrivers/tools used
  - [ ] Main line: 3/4" or 1" from facility compressor
  - [ ] Pressure: 80–100 PSI, filtered, regulated
- [ ] Data/Network: CAT6 drops to key stations (firmware loading station, documentation)
- [ ] Lighting: 500 lux minimum (critical for precision assembly work)

#### ESD Design
- [ ] Flooring: Conductive/dissipative ESD flooring throughout (10⁶–10⁹ Ω)
- [ ] Grounding: Central ground bus with branches to each workstation
- [ ] Work surfaces: All tables and fixtures bonded to ground

#### Safety & Ergonomics
- [ ] Workstation height: 30–36" adjustable (reduce fatigue, improve accuracy)
- [ ] Seating: Anti-fatigue mats for standing positions, ergonomic stools if sitting
- [ ] Lighting: Task lighting at each station (in addition to ambient lighting)
- [ ] Storage: Tool carts, component bins organized at each station
- [ ] Safety: Signage, emergency stops, fire extinguishers (every 75 ft)

### Construction/Installation Phase

#### Flooring & ESD Infrastructure
- [ ] ESD flooring installation (conductive or dissipative)
  - [ ] Resistance verification: 10⁶–10⁹ Ω (ANSI/ESD 7.1)
  - [ ] Sealing: All seams sealed, no exposed joints
- [ ] Central grounding bus (copper, ≥6 AWG)
  - [ ] Location: Run along one side of assembly floor or overhead
  - [ ] Test points: Installed every 20–30 ft for verification
  - [ ] Bonding conductors: From central bus to each workstation (≥18 AWG)
- [ ] Ground resistance verification: <1 Ω facility main ground

#### Workstations (8–15 stations, typical configuration)
Each workstation includes:
- [ ] Assembly table (48" W × 30" D × 34" H, height-adjustable if possible)
- [ ] Storage shelving above table (for tools, fasteners, small components)
- [ ] Component bins/trays (organized by assembly step)
- [ ] Documentation stand (vertical mount for work instructions, assembly diagrams)
- [ ] Test equipment access (multimeter, continuity tester on cart or nearby)
- [ ] Tool kit (screwdrivers, pliers, wrenches, specialty tools per assembly step)
- [ ] ESD mat (bonded to ground)
- [ ] Wrist strap attachment point

#### Material Handling & Staging
- [ ] Entry staging area
  - [ ] Rack receiving stand (holds frame upright for assembly)
  - [ ] Kitted component staging (totes with all parts for assembly)
- [ ] Inter-station material carts (3–4 mobile carts)
  - [ ] Size: Accommodate sub-assemblies, cables, components in process
  - [ ] Wheels: Heavy-duty, smooth-rolling
- [ ] Exit staging area
  - [ ] Cooling racks (allow completed assemblies to cool/stabilize before burn-in)
  - [ ] Wrapping/protection (bubble wrap or foam)
- [ ] Rework staging area
  - [ ] Designated space for units requiring correction
  - [ ] Documentation of rework reason

#### Electrical & Power Distribution
- [ ] Main distribution panel: 480V, 200+ amp service
- [ ] Step-down transformer (if 208V/120V needed): 75–100 kVA capacity
- [ ] Branch circuits:
  - [ ] Tool power: 20 amp × 8 circuits (pneumatic/electric tools)
  - [ ] Computer/lighting: 15–20 amp × 6 circuits
  - [ ] Emergency stop button: Accessible from multiple locations
  - [ ] Circuit breakers: Labeled, color-coded

#### Pneumatic System (if tools used)
- [ ] Main air line: 3/4"–1" copper or aluminum tubing
- [ ] Pressure regulator: Set to 80–100 PSI
- [ ] Moisture trap & filter: Remove water and contaminants
- [ ] Branch lines: 1/2" to each tool station
- [ ] Quick-connect couplers: Standard size for ease of tool swapping

#### Lighting
- [ ] Ambient lighting: 300–400 lux (high-bay fixtures)
- [ ] Task lighting: 500 lux at each workstation (LED strips or track lights)
- [ ] Lighting circuits: Separate from equipment power (avoid voltage fluctuations)
- [ ] Emergency lighting: Battery-backed, 10% of normal illumination level

#### HVAC & Climate Control
- [ ] Temperature control: Maintain 18–27°C (64–81°F)
- [ ] Humidity control: Maintain 40–60% RH
- [ ] Air distribution: Supply air to assembly area, return air from separate path
- [ ] Sensor placement: Temperature/humidity logger (data collection for compliance)

### Pre-Operations Phase

#### Equipment & Tools
- [ ] Workbenches: 8–15 units, all properly grounded
- [ ] Pneumatic tools:
  - [ ] Impact driver/screwdriver: 2–3 units per shift
  - [ ] Angle grinder (if needed): 1–2 units
  - [ ] Other specialized tools (per assembly steps)
- [ ] Hand tools:
  - [ ] Screwdrivers (multiple types/sizes): 2 sets per workstation
  - [ ] Pliers, wrenches, mallets: 1 set per workstation
  - [ ] Flashlights, magnifying glasses: For internal inspection
  - [ ] Torque wrenches (if torque specs required): 1–2 units
- [ ] Test equipment (mobile carts):
  - [ ] Multimeter: 2–3 units
  - [ ] Continuity tester: 2–3 units
  - [ ] Power analyzer (for plug-and-play verification): 1–2 units
