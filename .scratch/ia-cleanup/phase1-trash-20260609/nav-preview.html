What this reference proves

Built entirely on the production design system from Issue #01 — same tokens, same mega-menu components, same main.js behavior. Nothing here is a parallel system.

  • Capability tabs link to their hub and open a services mega-menu (hover on desktop, accordion on mobile).
  • Industries lists all 10 verticals; AI & Server Rack is also a featured top-level shortcut.
  • No division-switcher; one consistent header. RFQ is a persistent coral button (ink text, never white-on-coral).
  • CNC copy honors ADR-0002: a managed Monterrey network, never owned machines.
