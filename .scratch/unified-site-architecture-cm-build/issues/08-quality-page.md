# 08 — Quality page

Status: ready-for-agent
Parent: ../PRD.md

## What to build

Build a shared Quality page. Cover GPW's QMS, how it governs quality across the machining network (incoming and first-article inspection, certificate of conformance, material certs, lot traceability), in-house Assembly & Integration QA/test, and a dated ISO 9001 roadmap mapping AS9100D / ISO 13485 / IATF 16949 to the target industries. State in-place vs in-progress explicitly; no fabricated certifications.

## Acceptance criteria

- [ ] Quality page exists and is a top-level nav item.
- [ ] Describes the QMS, network quality governance, and in-house assembly QA.
- [ ] ISO 9001 roadmap with dates; cert standards mapped to industries.
- [ ] No claimed certifications that don't exist; honest in-place/in-progress labeling.
